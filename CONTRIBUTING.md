# Contributing to CloudExpat GitHub Action Example

Thank you for your interest in contributing!

## Reporting Issues

If you encounter any issues:

1. Check [existing issues](https://github.com/cloudexpat/cloudexpat-github-action-example/issues) first
2. If not found, [create a new issue](https://github.com/cloudexpat/cloudexpat-github-action-example/issues/new)
3. Include your workflow configuration (with secrets redacted)
4. Include any error messages from the GitHub Actions logs

## Suggesting Improvements

We welcome suggestions! Please open an issue describing:

- The improvement you'd like to see
- Why it would be useful
- Any implementation ideas you have

## Pull Requests

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/my-improvement`)
3. Make your changes
4. Test your changes by running the workflow
5. Submit a pull request
